# Jihye — Frontend Developer Portfolio

레트로 브라우저 스타일의 개인 프로필 사이트입니다.

## Tech Stack

- HTML5
- CSS3 (Custom + Tailwind CSS CDN)
- Vanilla JavaScript
- Font Awesome Icons

## Project Structure

```
├── index.html      # 메인 페이지
├── css/
│   └── style.css   # 커스텀 스타일
├── js/
│   └── main.js     # 인터랙션 스크립트
└── assets/         # 이미지 등 정적 리소스
```

## Getting Started

별도의 빌드 과정 없이 `index.html`을 브라우저에서 열면 바로 확인할 수 있습니다.

```bash
# Live Server 등을 사용하면 편리합니다
open index.html
```
